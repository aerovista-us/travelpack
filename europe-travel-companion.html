Good news: this page works without internet once saved on the device because everything is built into one file and stored locally.
Best practice: keep one copy on the laptop, one copy on the phone files app, and one exported JSON backup in email or cloud storage.
Before travel: save the file, open it once, enter the basics, then export a trip file after each major update.
For a printed backup: use the Print / Save PDF button and keep a simple paper copy of contacts, first hotel, passport info, and flight numbers.
